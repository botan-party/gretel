# sureta

## Requirements

- RAM: 4GB

## How to use

for Amazon Linux 2

### Install

```sh
sudo yum install -y git
git clone https://github.com/shokkunrf/sureta.git
cd ./sureta
bash ./setup.sh install <Minecraft Version>
```

### Run server

```sh
# start
sudo systemctl start spigot
# stop
sudo systemctl stop spigot
```
